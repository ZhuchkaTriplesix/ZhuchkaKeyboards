# Снимки OpenAPI (`*.openapi.json`)

Сюда по согласованию команды можно коммитить выгрузки `GET /api/openapi.json` из субмодулей сервисов для истории и диффов.

Именование: `<короткое-имя-сервиса>.openapi.json` (например `auth.openapi.json`, `catalog.openapi.json`).

Процедура: [../../openapi-sync.md](../../openapi-sync.md).
